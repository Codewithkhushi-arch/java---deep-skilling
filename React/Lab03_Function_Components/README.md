# Lab 3: Function Components

## Overview
This lab covers the creation of a functional component in React. Functional components are simpler, more concise, and are the standard approach in modern React development alongside Hooks.

## Java Developer Perspective
**Functional Components vs. Java Static Methods / Functional Interfaces**
A React Functional Component is analogous to a stateless static method in Java that takes arguments (props) and returns a value (JSX). 
Like Java 8's Functional Interfaces (e.g., `Function<T, R>`), a React function component cleanly maps input data to output UI without the boilerplate of a class, inheritance, or instance management.

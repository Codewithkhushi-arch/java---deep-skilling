# Java Developer Portfolio

Welcome to my Java Developer Portfolio! This repository showcases a comprehensive collection of my backend and full-stack projects, demonstrating proficiency in the Java ecosystem, modern web frameworks, and software engineering best practices.

## 🚀 Skills & Technologies

*   **Core Java:** Java 11/17, Object-Oriented Programming, Data Structures, Algorithms, Design Patterns.
*   **Spring Ecosystem:** Spring Boot 3, Spring Core, Spring MVC, Spring Data JPA.
*   **Microservices:** Spring Cloud (Eureka, API Gateway, OpenFeign, Resilience4j, Config Server).
*   **Database & ORM:** SQL (PL/SQL), Hibernate, Relational Database Design.
*   **Testing:** JUnit 5, Mockito, Spring Boot Test, SLF4J (Logging).
*   **Frontend:** ReactJS, Component-based UI design.
*   **Tools & DevOps:** Git, Maven, Postman.

## 📁 Repository Structure

*   `Spring_REST_using_Spring_Boot/`: RESTful APIs built with Spring Boot, including custom exception handling, JWT-based security, and CRUD operations.
*   `Microservices/`: Distributed system architectures implementing service discovery, API gateways, centralized configuration, and resilient communication.
*   `Spring_Data_JPA_with_Hibernate/`: Advanced data access layers demonstrating entity mappings, complex queries, and transaction management.
*   `Spring_Core_and_Maven/`: Foundational projects highlighting Inversion of Control (IoC), Dependency Injection (DI), and Aspect-Oriented Programming (AOP).
*   `JUnit_Mockito_and_SL4J/`: Comprehensive test suites utilizing TDD practices and mock-driven testing.
*   `Design_Patterns/`: Implementations of standard GoF design patterns (Singleton, Factory, Observer, etc.) in Java.
*   `React/`: Frontend web applications showcasing integration with backend REST APIs.
*   `SQL/`: Database schemas, queries, and PL/SQL scripts.
*   `DSA/`: Implementations of core data structures and algorithmic problem solving.
